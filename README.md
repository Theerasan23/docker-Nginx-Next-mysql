
## Installation

Install project  with npm
 -  install docker 


 /api/.env
```bash
SERVER_HOST="localhost"
SERVER_PORT="3010"

# mysql 

DB_HOST="ip"
DB_USER="root"
DB_PASSWORD="pass"
DB_DATABASE="database"
DB_PORT=65514
```

then
```bash
  docker-compose up -d
```

or testing nextjs
```bash
  npm i  or npm init 
  npm run dev
```
